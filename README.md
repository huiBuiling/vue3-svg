## Vue 3 + TypeScript + Vite

```sh
# 1. 克隆项目至本地
git clone xx

# 2. 安装项目依赖
yarn install

# 3. 运行项目
yarn dev
```

- 可视化组件配置生成不同头像

- 撤销，还原
- 下载
- 国际化语言
